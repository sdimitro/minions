Programming Language: Python (version 3.5)
Libraries/Packages: NumPy (version 1.10)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

* Instructions

To run the program:
`$ python project.py` or `$ make`

To play with the program data and functions:
`$ python -i project.py` or `$ make repl`

To output the dependencies:
`$ make deps`

To attempt to run program with an older version of Python:
`$ PYTHON=python3.4 make` (same for make repl)

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

* Notes

You can try running the program with older
versions of Python but I can't guarantee it
will work.
